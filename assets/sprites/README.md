# Sprites

Будущие PNG/spritesheet ассеты. Текущая версия игры использует generated placeholders из `src/main.js`.

Ожидаемые имена для быстрой замены:

- `dino-run-0.png`
- `dino-run-1.png`
- `dino-run-2.png`
- `dino-jump.png`
- `dino-slide.png`
- `obstacle-cactus.png`
- `obstacle-hurdle.png`
- `obstacle-cone.png`
- `h2o-bottle.png`
- `shop-stand.png`
